Sobre os meios de leitura digital.
- 56% dos leitores utilizam smartphones.
- 49% dos leitores utilizam computador.
- 18% dos leitores utilizam tablet.
- 4% dos leitores utilizam e-reader.
Os e-readers são bastante indicados para ambientes externos principalmente durante o dia. Os e-readers que possuem iluminação própria também são ótimas escolhas para se utilizar para leitura antes de dormir.
